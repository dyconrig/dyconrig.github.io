DyConRIG: Dynamic Consensus for Robotic Information Gathering

DyConRIG is a distributed multi-robot information gathering system that employs a consensus-based control strategy to efficiently map environmental fields. By integrating a modified Hegselmann-Krause model, it enables robots to balance local and shared information, prioritizing uncertainty-driven sampling of informative locations. Validated through simulations and real-world experiments, DyConRIG achieves robust, scalable, and efficient environmental monitoring with asynchronous robot coordination.

Key Contributions

The DyConRIG system introduces several advancements in multi-robot information gathering:

  • Consensus-Based Control: Integrates a modified Hegselmann-Krause model to balance local and shared information, enabling optimal control decisions.
  • Uncertainty-Driven Sampling: Prioritizes visits to unexplored, informative locations using Gaussian Process models with attentive kernels.
  • Decentralized Coordination: Enables asynchronous robot motion and task allocation, enhancing scalability and robustness.
  • Real-World Validation: Demonstrated through simulations and field experiments with Autonomous Surface Vehicles, achieving efficient environmental mapping.

Autonomous Surface Vehicle Experiments

The DyConRIG system was implemented for Autonomous Surface Vehicles (ASVs) to map temperature fields in a 2D aquatic environment. Experiments were conducted across eight distinct environments, with robots modeled as 5D state vectors (position, orientation, linear and angular velocities). DyConRIG successfully generated online temperature maps while avoiding inter-vehicle collisions, validated through both simulations and real-world tests.

Benchmark Results

DyConRIG was benchmarked using the Probabilistic Online Active Mapping (POAM) method across eight environments (N17E073, N18E045, N35W107, N42E054, N43W080, N44W111, N45W123 stipend N47W124) with varying terrain complexities from NASA's Shuttle Radar Topography Mission (SRTM) elevation maps. Performance was evaluated using Standardized Mean Squared Error (SMSE), Mean Standardized Log Loss (MSLL), Negative Log Predictive Density (NLPD), and Root Mean Square Error (RMSE). The Hegselmann-Krause (HK) Consensus planner with POAM outperformed other methods, achieving robust performance (e.g., RMSE 8.88 ± 2.26 in N43W080 with α=0.3). The stubbornness parameter (α) was optimized by selecting values with the lowest upper confidence bound (mean + standard deviation) for RMSE, ensuring robust performance across diverse terrains. An analysis of team size (3, 4, 5 robots) in N43W080 further demonstrated that smaller teams (e.g., 3 robots) often achieved superior performance due to reduced coordination overhead, with results evaluated using the upper confidence bound.
